# Inspiracje

## 1. Inspiracje (słowa, skojarzenia, pomysły)
- prostota
- minimalizm UI
- szybkie wyszukiwanie
- personalizacja
- kolorowe kategorie
- motyw edukacyjny
- gamifikacja (punkty, odznaki)
- responsywność
- integracja z kalendarzem
- tryb offline

## 2. Krótka lista elementów projektu
- strona główna z panelem nawigacyjnym
- system kategorii i tagów
- moduł wyszukiwania i filtrowania
- ekran szczegółów elementu
- panel ustawień użytkownika
- baza danych (lokalna/serwer)
- mechanizm synchronizacji (opcjonalnie)
- testy jednostkowe podstawowych funkcji

## 3. Wstępne dane wejściowe (np. lista kategorii, lista funkcji)
- lista kategorii: "Inspiracje", "Projekty", "Zadania", "Materiały", "Pomysły"
- pola elementu: tytuł, opis, kategoria, tagi, data utworzenia, priorytet, status
- funkcje aplikacji: dodaj/edytuj/usuwaj element, przeszukaj, filtrowanie po kategorii i tagach, eksport/import (CSV), oznacz jako ukończone, przypomnienia
- ustawienia użytkownika: motyw (jasny/ciemny), język, opcje synchronizacji, powiadomienia

## 4. Lista źródeł pomysłu
- własna potrzeba organizacji pomysłów
- inspiracja od znajomego pracującego nad UX
- istniejące aplikacje typu "notes" i "task manager"
- przypadkowy artykuł o gamifikacji i produktywności
